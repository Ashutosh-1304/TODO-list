# 📝 TODO List App

A simple and interactive Todo List web application built using **HTML, CSS, and Vanilla JavaScript**.  
This project demonstrates core concepts of DOM manipulation and event handling.

---

## 🚀 Features

- ✅ Add new tasks  
- ✅ Delete tasks by clicking on them  
- ✅ Prevent empty task submission  
- ✅ Clean and minimal UI  
- ✅ Responsive layout  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla JS)** – Functionality  

No external libraries or frameworks used.


---

## ⚙️ How It Works

1. User enters a task in the input field.
2. Clicks the **Enter** button.
3. A new `<li>` element is dynamically created.
4. The task is appended to the list.
5. Clicking on a task removes it.
6. Empty inputs are ignored using `.trim()` validation.

---

## 🔍 JavaScript Concepts Used

- `getElementById()`
- `addEventListener()`
- `createElement()`
- `appendChild()`
- `remove()`
- `.trim()` for input validation
